# AI API中转站推荐与评测

**2026-3-20更新**, 目前这个阶段, 只写代码和用GPT, 不想折腾又划算的方案是
1. 在闲鱼上买一个 Codex team拼车的号, 一般是几块钱人民币保一个月
2. 然后把邮箱给他. 
3. 你的邮箱会收到一封邮件. 邮件里有一个邀请链接.
4. 点击邀请链接后, 用这个邮箱登录chatgpt, 就有 Codex team权限, 可以使用GPT5.4
5. 如果要用 Codex 写代码, 在目录下执行codex命令, 选择网页认证.
6. 如果要养小龙虾,也可以用这个账号. 基本够用了

----

随着龙虾火起来之后, 大家都需要购买API KEY. 但是API中转站水很深, 非常看站长的良心

在国内想要安心的使用最先进的AI API 真是太难了. 国外的AI不仅价格贵, 而且经常被误杀, 很多时候都会误事.

选择一个好的中转站不容易, 要看下面的这些东西

1. 最重要的是稳定，如果它自己经常宕机或延迟极高，那不仅没省事反而耽误事. 所以要优先买那些信誉好,运行时间长, 最好有群的站
2  第二个是关注服务器节点位置。对于国内开发者，如果中转服务器在香港、新加坡, 就比在美国的站点好很多.
3. 模型的覆盖性: 一个好的中转站应该让你一站式调用全球顶尖模型, 最好能尽快上官方的最新模型.
4. 中转站的收费有很多问题. 很多站看起来价格低, 实际运行起来价格很高. Token计数, 或者价格费率, 都是造成价格不同的很重要原因.  另外站点最好要有清晰的账单, 否则将来哪一天突然账单异常了,都查不到原因. 也要警惕价格异常便宜的模型, 大概率是拿着GLM这种低价国产模型代替高价的国外模型.
5. 每个站都有跑路的危险, 优先选择公司运营的站点. 还有他们客服响应问题的速度, 站点速率波动时有没有通知这些.

为了解决下面的问题, 我对使用过的中转站做一个评价.

在所有使用之前, 切记一点, 这个行业不稳定, 不要大额充值, 用多少充多少.

## 推荐的

### OpenRouter.ai
OpenRouter在所有的网站里, 体验是最好的. 
一个API可以使用多个模型,  最新的GPT API, Anthropic API, Gemini , 国内的DeepSeek, Kimi, MiniMax都可以第一时间使用.

稳定性非常不错,  而且都是原厂模型, 直接使用就行.

缺点是价格比较贵, 很多情况下比原厂价格还要贵5%.  另外, 虽然OpenRouter承诺不存储请求数据, 但是多一手毕竟是多一份风险.

### [PackyCode](https://www.packyapi.com/register?aff=gF1p)
PackyCode 大约是24年底25年初开始活跃, 是国内比较早针对Claude Code进行优化的供应商.他与开源社区互动比较频繁, 站长在x上也非常活跃, 客服响应比较快. 随着发展, 这个站已经是很多中转站的上游供应商.

反代出来的Claude Sonnet4.6 大概是0.3折左右, 折合下来约等于不要钱, 人民币¥0.45/¥2.25一百万Token.(以下都以人民币为单位)
Opus 4.6 反代出来的价格是 人民币¥0.75/3.75 一百万Token.

Codex现在是2.5折. 支持包月套餐, 最便宜的是45人民币一个月, 对应每周90美元的额度. 最贵的是90人民币一个月, 对应每周360美元额度.

他们对于国内的模型支持比较全, 支持阿里百炼的api, MiniMax(官方价格的5折),GLM(九折). 

新人注册送1美元, 可以先试试再决定购买.

上新模型也比较快, 更多的是面向企业的服务, 稳定性挺好, 还可以开发票.

有美国和国内的高防域名.

可以使用Claude code专用组, Opus4.6的价格是人民币¥6/30 一百万Token. 我自己测试了下, 感觉没掺水.
GPT 5.4是人民币¥1/6一百万Token. 价格是不便宜.

追求稳定的企业客户可以上.

新用户送一点钱,每天签到还能送点钱, 值得试试.

## 不推荐的

### 闲鱼上的中转站
试过好几个, 闲鱼上不知名中转站, 基本要不是模型掺水的特别厉害, 要不就是速度不达标, 太看运气了.


### [aicodemirror](https://www.aicodemirror.com/register?invitecode=7C06QL)
aicodemirror 是一个老牌的中转站, 之前使用的人很多
目前支持claude, gpt, gemini 三种api. 
价格方面,claude是官方渠道是3.8折, 如果是反代出来的接口目前是1.9折. codex和gemini很便宜, 分别是0.6折和1折.

但是最近这个网站被很多朋友爆料使用他们使用别的模型掺水, token消耗很快, 并且客服服务态度很差. 不推荐使用这家.


### [大肘子](https://api.dzzi.ai/register?aff=AUIz)
大肘子又是一个NewAPI搭建的API 中转站

这个站在xhs上非常火, 大家的反馈都很好. 这个站反代的非常明显, 特点是按调用次数收费, 而不是常见的按量收费.
譬如, 有1毛钱一次调用的opus4.6接口, 也有3毛的. 理论上3毛应该比1毛的质量要好点.(这个渠道怎么做到的, 我有点疑问)

新用户注册会送0.5元, 每天还可以签到. 对于一些便宜模型, 应该可以请求好几次. 
而且时不时还会一些免费的模型可以用.

但是这个按次其实还是贵的, 譬如这个网站里,gpt5.4是5分一次请求, 但是在很多上面推荐的网站, 基本平均一次请求只要2-3分.

只允许qq邮箱注册, 售后QQ群目前到了6群, 群里比较活跃.

因为编程时一个任务一般需要多次请求, 所以这个站不适合编程使用. 而在二次元场景下适合一些. 



### [柏拉图AI](https://api.bltcy.ai/register)
柏拉图好像是一个比较老的API站, 2024年就有了.
他的优点是支持市面上基本上所有的模型, 除了常见的OpenAI, Claude, Gemini这三家外, DeepSeek, 豆包, MiniMax, QWen,GLM这些也支持, 他们还支持  Mid-journey(文生图),suno_music(文生歌), 快手可灵(文生视频),Ideogram(文生图)等等. 可能别的地方找不到的接口, 都可以在这里找到.
这一点非常厉害.

另外, 这个网站除了大陆外, 还有美国,香港的接入点,还不错. 接口的响应也挺快.

用户注册送2毛钱, 每天签到能送一点钱.

价格方面, Claude Opus4.6是人民币¥5/25 一百万Token, Sonnet4.6是人民币¥3/15一百万token. 但是我试用了一会, 发现用低级别模型掺水挺严重的.不知道别的朋友试用效果如何. 

GPT5.4是人民币¥2.5/15 一百万Token.


不支持月卡.

如果要买便宜的,比他便宜的多的是, 可以选择别的.
如果要买支持各种模型的, 可以考虑这个站.


### [LingxiCode](https://new.050602.xyz/register)
LingxiCode 是2025年年底开的一个站. 支持Claude, Gemini, GPT, DeepSeek, Kimi, GLM, MiniMax这些常用的.

价格方面比较便宜, 最便宜的Sonnet4.6 是¥0.9/4.5 一百万token,  最便宜的Opus 4.6的价格是人民币 ¥5/25 一百万token.
但是接口质量一般.

GPT5.4价格很贵, 人民币¥1.37/8.25 一百万token.

还有, 为什么用一个这么便宜, 这么难记的域名..
